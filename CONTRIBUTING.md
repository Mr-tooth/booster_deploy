# Contributing

## Development Environment
- Recommended conda env: `booster_deploy`
- Python: `3.10+`

## Common Commands
- `make ruff`: run ruff static checks.
- `make lint`: run lint + bytecode compile checks.
- `make format`: apply ruff formatting.
- `make test`: run unit tests.
- `make check`: run lint and tests together.

## Notes
- Run commands from repository root.
- If `ruff` is missing, install it into the active environment:
  - `python -m pip install ruff`
